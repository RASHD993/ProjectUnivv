# Home Essentials Store

  A React + Vite home goods store. No backend required — all data is built in.

  ## Getting Started

  ```bash
  npm install
  npm run dev
  ```

  Open http://localhost:5173 in your browser.

  ## Stack
  - React 18
  - Vite
  - Tailwind CSS v4
  - Wouter (routing)
  - React Hook Form + Zod (checkout form)
  - Radix UI primitives (accessible components)
